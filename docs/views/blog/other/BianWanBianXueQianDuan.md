### 使用 JS/TS 代码控制自己的殖民地

[Screeps](https://screeps.com/)

Screeps 是一款面向编程爱好者的开源 MMO RTS 沙盒游戏，其核心机制是为您的单位编写AI。您可以通过编写 JavaScript 来控制自己的殖民地。

Steam售价：65元

建议先上官网试玩，不要直接用steam客户端游戏，省得记录了时间，被劝退时不好退款。玩这个游戏需要阅读大量英文API以及一定编程能力
