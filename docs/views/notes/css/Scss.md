### 深度选择器
优雅的只在当前页面中覆盖ui库中组件的样式
``` scss
.van-tabs ::v-deep .van-ellipsis { color: blue};
//编译前

.van-tabs[data-...] .van-ellipsis{
    color:blue;
}
//编译后
```
